# Vision

## Community Identity Platform

Member Manager は BNI 専用システムではなく、
将来的には汎用 Community Identity Platform として展開可能な構造を目指す。

## 想定対象

- BNI
- 一般コミュニティ
- 協会
- オンラインサロン
- 店舗グループ
- 研究会
- 業界団体

## 中核設計

- Organization Hierarchy
- Role Based Access Control
- Scoped Permission
- Permission Inheritance
- Direct Permission
- deny Permission
- Auditability
- Historical Integrity
